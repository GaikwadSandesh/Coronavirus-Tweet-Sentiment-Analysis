# Coronavirus-Tweet-Sentiment-Analysis\
Abstract: During the time of pandemic to manage and keep law and order it is important to know what kind of sentiments people are having. It also helps to target people who are misinformed about the pandemic and help them.

This Ml project uses tweets from different parts of the world and builds models to  help them sort according to the sentiments of tweets.

Keywords: machine learning, ordinal error, NLP

Problem Statement:  Use the tweet data and build a machine learning model that will classify the tweet sentiments.
